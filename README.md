# 🚀 SaaS Landing Page — Novu AI

A high-converting, premium SaaS landing page for an AI meeting assistant product. Every section a startup needs to go from idea to launch-ready — designed to explain, persuade, and convert.

---

## 📄 Sections

| Section | Description |
|---|---|
| **Navigation** | Fixed top nav with section links, Sign in link, and a Start free CTA button |
| **Hero** | Bold headline, subtext, two CTAs, trust badges (free forever, no credit card, Zoom & Meet support), and a live animated product demo card showing a real-time transcript and generated summary |
| **Social Proof Bar** | Logo strip of trusted companies — Stripe, Figma, Linear, Vercel, Loom, Notion, Intercom |
| **Problem Section** | Three-card breakdown of the problem with animated counters (0% notes, 0% action items completed, 0hrs wasted) |
| **Features** | Four feature cards — Auto-join, Smart summaries, Action item extraction, Slack & Notion integration |
| **How It Works** | Three-step visual walkthrough with a live transcript animation on the right side |
| **Integrations** | Icon grid of supported platforms — Zoom, Google Meet, Slack, Notion, Google Calendar, MS Teams |
| **Testimonials** | Three star-rated quotes from Product Manager, Engineering Lead, and Head of Ops personas |
| **Pricing** | Three-tier pricing cards — Free ($0), Pro ($12/mo, highlighted), Team ($39/mo) — with feature lists and CTA buttons |
| **FAQ** | Six accordion questions covering recording, participant visibility, language support, Teams, data security, and free plan limits |
| **Final CTA** | Full-width closing section with headline and start free button |
| **Footer** | Four-column footer — Product, Company, Support links, and social icons |

---

## 🎨 Design

- Dark background with blue/violet accent gradient — modern SaaS aesthetic
- Animated product demo in the hero that simulates a live transcript being generated
- Animated stat counters in the Problem section that count up on scroll
- Three-step How It Works section with a live-looking chat animation
- Middle pricing tier highlighted with a "Most popular" badge
- Accordion FAQ with smooth open/close animation
- Fully responsive — all sections reflow cleanly on mobile

---

## ✏️ How to Customise

To swap in a client's product, update:

- **Product name and tagline** — hero, navbar, and footer
- **Hero demo card** — transcript names, summary text, and action items
- **Social proof logos** — company names in the logo bar
- **Problem stats** — the three counter values and descriptions
- **Feature cards** — icons, titles, and descriptions
- **How it works steps** — three step titles and descriptions
- **Pricing tiers** — names, prices, and feature lists
- **FAQ items** — questions and answers
- **Testimonials** — names, roles, companies, and quotes

All content is in plain HTML — no build step needed.

Upload the folder to any static host. Works on Vercel, Netlify, GitHub Pages, or any standard web host.
